📋 DESCRIPTION
--------------
Plateforme web complète pour bureau d'études environnemental en Algérie
avec site vitrine, veille réglementaire et espace client sécurisé.

🎯 OBJECTIFS ATTEINTS
---------------------
✅ Site vitrine avec expertise réglementaire algérienne (Loi 03-10)

🚀 DÉMARRAGE RAPIDE (5 MIN)
---------------------------

1. INSTALLER POSTGRESQL
   > sudo apt install postgresql
   > sudo systemctl start postgresql

2. CRÉER LA BASE DE DONNÉES
   > sudo -u postgres psql
   > CREATE DATABASE environmental_db;
   > \q
   > sudo -u postgres psql environmental_db < server/database/schema.sql

3. CONFIGURER LE BACKEND
   > cd server
   > npm install
   > cp .env.example .env
   > nano .env  # Éditer DB_PASSWORD
   > npm run init-db
   > npm start  # Port 5000

4. DÉMARRER LE FRONTEND
   > cd ../client
   > npm install
   > npm run dev  # Port 3000

5. SE CONNECTER
   URL: http://localhost:3000/login
   Email: admin@bureau-etudes.dz
   Password: Admin123!
